# Obsitrail.Result

Minimal ve güvenli bir **Result / Error kütüphanesi**  
Hataları exception fırlatmadan nesne olarak taşır, kodun okunabilirliğini ve test edilebilirliğini artırır.

---

## Özellikler

- `Result<T>` ile başarılı veya başarısız işlemleri temsil eder
- `Error` sınıfı ile hata mesajını taşır
- Testlerle doğrulanmış (%100 Passed)
